# bitMap

побитовые настройки параметров  переменной для **type**=[**16BitXX**, **32BitXXXX**]

|Свойство|Значение|
|----|---:|
|Тип|JSON|
|Обязательный|Нет|

----

## bitMap.bit

Номер бита переменной

|Свойство|Значение|
|----|---:|
|Тип|int|
|Обязательный|Да|
|Значение для **type**=**16BitXX**|0...15|
|Значение для **type**=**32BitXX**|0...31|
