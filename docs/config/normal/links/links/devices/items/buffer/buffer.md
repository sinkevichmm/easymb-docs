# **buffer**

Настройки параметров буферизации значений переменной

|Свойство|Значение|
|----|---:|
|Тип|JSON|
|Обязательный|Нет|

----

## **buffer**.enable

разрешение на буферизацию данных

|Свойство|Значение|
|----|---:|
|Тип|bool|
|Обязательный|Нет|
|Значение|true/false|
|Значение по умолчанию|false|

----

## **buffer**.size  

Глубина буферизации переменной

|Свойство|Значение|
|----|---:|
|Тип|int|
|Обязательный|Нет|
|Значение|0...255|
|Значение по умолчанию|0|
