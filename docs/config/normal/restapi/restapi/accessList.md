# accessList

Список разрешенных ip адресов клиентов с соответствующими правами доступа

|Свойство|Значение|
|----|---:|
|Тип|массив JSON|
|Обязательный|Да|

Може быть переопределен параметром **links**[x].**devices**[x].**items**[x].**remoteAccess**.**restapi**.**accessList**

----

## accessList.ip

адрес клиента

|Свойство|Значение|
|----|---:|
|Тип|string|
|Обязательный|Да|
|Значение|"ddd.ddd.ddd.ddd"|

----

## accessList.read

разрешение на чтение переменной

|Свойство|Значение|
|----|---:|
|Тип|bool|
|Обязательный|Нет|
|Значение|true/false|
|Значение по умолчанию|false|

----

## accessList.write

разрешение на изменение переменной

|Свойство|Значение|
|----|---:|
|Тип|bool|
|Обязательный|Нет|
|Значение|true/false|
|Значение по умолчанию|false|
