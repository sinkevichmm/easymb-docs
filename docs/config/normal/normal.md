# **Обычный режим**

Это основной режим работы приложения. Предназначенный для штатной работы системы.

```shell
# easymb --config path/to/config/file.json
```

Так же это режим можно запусатить с параметром --monitor.

```shell
# easymb --config path/to/config/file.json
```
